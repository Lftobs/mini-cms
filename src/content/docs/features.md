# Features

Mini CMS is packed with features designed to make content management seamless for developers and content creators alike.

## GitHub Integration

Directly connected to your GitHub account. No middleman database. Your content is your code.

- Automatic branch management (`mini-cms-flow`).
- Commit-based saving.
- Support for private and public repositories.

## MDX & Markdown Editor

A powerful editor that handles both standard Markdown and MDX.

- **WYSIWYG Editing:** See your changes as you type.
- **Component Support:** Insert and edit JSX components in your MDX files.
- **Image Uploads:** Upload images directly to your repository.

## Organization Management

Collaborate with your team easily.

- Invite team members to your organization.
- Role-based access control (Owners and Members).
- Manage multiple projects under one roof.

## Activity Tracking

Keep an eye on what's happening in your projects.

- Audit logs for all file changes.
- Contributor tracking.
- Project-wide activity feed.

## Modern Tech Stack

Built with the latest technologies for speed and reliability.

- **Astro:** Blazing fast performance.
- **Hono:** Lightweight and fast API.
- **Alpine.js:** Reactive UI elements.
- **Tailwind CSS:** Beautiful, responsive design.
